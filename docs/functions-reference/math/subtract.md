# Subtract
Referred as `#subtract`

## Description
The `subtract` function subtracts `B` from `A`.

## Inputs
| Name | Type |
|------|------|
| `A` | `number` |
| `B` | `number` |

## Outputs
| Name | Type |
|------|------|
| `result` | `number` |
| `errorMessage` | `string` |

The `errorMessage` property will only be defined if any of B or A are not numbers.
