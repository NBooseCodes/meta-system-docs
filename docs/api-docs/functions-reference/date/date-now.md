# Date Now
Referred as `#dateNow`

## Description
The `dateNow` function has no inputs, it simply returns the current date .

## Inputs
> This function has no inputs


## Outputs
| Name | Type |
|------|------|
| `now` | `date` |

## Examples
| <h2> Outputs </h2> |
|:-----:|
| **now** |
| `Wed Sep 01 2021 22:39:35 GMT-0300 (Brasilia Standard Time)` |
